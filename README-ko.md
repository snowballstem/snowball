# Snowball 한국어 안내

이 문서는 이 저장소에서 한국어 스테머(korean stemmer) 관련 변경 사항을 간단히 설명합니다.

## 한국어 스테머 동작 요약

- 조사/어미 중심의 경량 규칙 기반 스테밍을 수행합니다.
- 길이가 2글자 이하인 토큰은 스테밍하지 않습니다.
- 고유명사 예외로 `한반도`는 그대로 유지합니다. (`한반`으로 축약하지 않음)

## 예시

- `동아시아의` -> `동아시아`
- `남부에` -> `남부`
- `한반도` -> `한반도`

## 빠른 테스트 방법

```bash
printf "한반도\n남부에\n동아시아의\n" | ./stemwords -l korean -p2
```

위 명령은 원문/스테밍 결과를 2열로 보여줍니다.
